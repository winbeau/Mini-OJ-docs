## 判题机

真实判题(编译运行 C++/Python 提交)由独立的判题机承担,代码在 [`judge/`](judge/)(单文件 `judge.cpp`,编译型)。
Java 端通过 `ProcessBuilder` 调用它取结果,见 [11-附录-判题机集成](11-附录-判题机集成.md)。

## 项目最终形态

```
Swing 客户端 ──Socket──▶ 多线程判题服务端 ──JDBC──▶ MySQL
                              │
                              ▼
                  判题机(独立编译型程序,编译运行 C++/Python 提交)
```

> 判题核心后续由独立的「判题机」(C++ 单文件,编译运行真实提交)承担,Java 端通过调用它取结果。
