# 10 · 第13章 网络 C/S(里程碑 M6b,收官)

> 目标:拆成客户端/服务端——客户端用 `Socket` 远程提交,服务端用 `ServerSocket` 接收、每连接一线程判题、回写结果。
> 最终架构成立:**Swing 客户端 ↔ Socket ↔ 多线程判题服务端 ↔ MySQL**。

## 知识点(大纲对照)

| 要点 | 处理 |
|---|---|
| `Socket` / `ServerSocket` | **动手**(核心) |
| `InetAddress`:获取客户端地址 | **动手**(打印来源) |
| 每连接一线程(呼应 Ch12) | **动手** |
| `URL` 读远程资源 | **了解/可选**(远程拉题库) |

## 协议(极简,一行文本)

为避免"通过网络传 Java 对象/源码"的复杂度,教学版约定:**服务端内置若干解法**,客户端只发"题号 + 解法名 + 用户名"。

```
客户端 → 服务端:  "1 aplusb alice"          (空格分隔)
服务端 → 客户端:  "AC 2/2 (3ms)"            (JudgeResult.toString)
```
> 这样网络层只传字符串,简单清晰;判题仍复用前面所有后端代码。

## 0. 服务端内置解法注册表 `oj/judge/Solutions.java`

```java
package oj.judge;
import java.util.*;
public class Solutions {
    private static final Map<String, Solution> REG = new HashMap<>();
    static {
        REG.put("aplusb", new AplusB());
        // 以后加新解法在这里登记
    }
    public static Solution get(String name) { return REG.get(name); }
}
```

## 1. 判题服务端 `oj/net/JudgeServer.java`(ServerSocket + 每连接一线程)

```java
package oj.net;
import oj.core.*;
import oj.io.ProblemLoader;
import oj.judge.*;
import java.io.*;
import java.net.*;
public class JudgeServer {
    public static void main(String[] args) throws IOException {
        ServerSocket server = new ServerSocket(6789);
        System.out.println("判题服务端启动,监听 6789...");
        while (true) {
            Socket sock = server.accept();              // 阻塞等连接
            new Thread(() -> handle(sock)).start();     // 每连接一线程(呼应 Ch12;可换线程池)
        }
    }

    private static void handle(Socket sock) {
        try (Socket s = sock;                           // try-with-resources 自动关闭
             BufferedReader in = new BufferedReader(new InputStreamReader(s.getInputStream()));
             PrintWriter out = new PrintWriter(s.getOutputStream(), true)) {

            System.out.println("来自 " + s.getInetAddress().getHostAddress());  // InetAddress
            String line = in.readLine();                                        // "1 aplusb alice"
            if (line == null) return;
            String[] t = line.trim().split("\\s+");
            Problem p = ProblemLoader.load(Integer.parseInt(t[0]));
            Solution sol = Solutions.get(t[1]);
            if (sol == null) { out.println("CE 0/0 (未知解法)"); return; }

            JudgeResult r = new StandardJudge().judge(p, sol);
            out.println(r.toString());                                          // 回写结果
        } catch (Exception e) {
            System.err.println("处理连接出错: " + e.getMessage());
        }
    }
}
```
> 进阶(可选):把判题改成丢进 M6a 的 `JudgeQueue`,由工作线程池消费——和第12章彻底打通。教学版直接判也行。

## 2. 客户端 `oj/net/JudgeClient.java`(Socket)

```java
package oj.net;
import java.io.*;
import java.net.*;
public class JudgeClient {
    public static String submit(String host, int problemId, String solName, String user)
            throws IOException {
        try (Socket sock = new Socket(host, 6789);
             PrintWriter out = new PrintWriter(sock.getOutputStream(), true);
             BufferedReader in = new BufferedReader(new InputStreamReader(sock.getInputStream()))) {
            out.println(problemId + " " + solName + " " + user);   // 发请求
            return in.readLine();                                  // 收结果
        }
    }
    public static void main(String[] args) throws IOException {
        System.out.println(submit("localhost", 1, "aplusb", "alice"));  // 期望: AC 2/2 (...)
    }
}
```

## 3. 跑起来(两个终端)

```bash
# 终端1:起服务端
java -cp "build:lib/*" oj.net.JudgeServer
# 终端2:客户端提交
java -cp "build:lib/*" oj.net.JudgeClient
# 服务端打印 "来自 127.0.0.1",客户端打印 "AC 2/2 (...)"
```
